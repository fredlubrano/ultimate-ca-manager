2. Reach the UI:

{{- end }}

3. Log in:

   user: admin
{{- if .Values.initialAdminPassword }}
   password: (the value you set in initialAdminPassword)
{{- else }}
   password: changeme123   <-- CHANGE THIS IMMEDIATELY after first login
{{- end }}

IMPORTANT
- This chart runs a SINGLE instance (replicaCount=1). The bundled scheduler is
  not HA-safe and the default SQLite DB is single-writer. Do not scale up.
- The "{{ include "ucm.etcPvcName" . }}" PVC holds /etc/ucm/master.key, which
  decrypts every private key in the database. It is annotated to survive a
  `helm uninstall`. Back it up and never delete it while data exists.
{{- if eq .Values.database.type "sqlite" }}
- Using the bundled SQLite database (data PVC). For an external PostgreSQL set
  database.type=postgresql and database.databaseUrl (or database.existingSecret).
{{- end }}
